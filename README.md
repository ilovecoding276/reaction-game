Reaction Challenge Game
Description

Reaction Challenge Game is a simple browser game where the player waits for a box to turn green and clicks as fast as possible. The game measures reaction time, keeps track of score, lives, and a countdown timer. The goal is to react quickly and achieve the best reaction time before the time runs out.

How to Play
Wait for the red box to turn green
Click the box as fast as possible when it turns green
If you click too early, you lose a life
Try to get the fastest reaction time before the timer reaches zero
Controls
Mouse click: Click the box
Features
Reaction time measurement
Score tracking
Lives system
Countdown timer
Best time tracking
Restart button
Technology Used
HTML
CSS
